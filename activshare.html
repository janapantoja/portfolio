ACTIVSHARE

Challenge

We acknowledge everyone is busy, but we do not want anyone to miss out on opportunities for good if we can help it. To save you time, we will figure out what causes you care about and help you find charities doing great work. My team and I created ActivShare to help with two important goals: to stay active and to support the causes one cares about. This was a projects developed and presented at BattleHack-Raleigh. See some features below:

  • The user signs up with email or via Twitter
  • Machine learning algorithm suggests charity matches on JustGiving.com based on the user's Twitter timeline
  • The user starts a physical challenge, invites friends, and makes a pledge for a charity
  • Payments can be made via Paypal
  • Organizations engage with their donors via GotoWebinar, and the user is able to learn about his/her favorite charity's work.
