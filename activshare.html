ActivShare

Challenge

We acknowledge everyone is busy, but we do not want anyone to miss out on opportunities for good if we can help it. To save you time, we will figure out what causes you care about and help you find charities doing great work. My friends and I created ActivShare to help with two important goals: stay active and support the causes one cares about. This was a projects developed and presented at BattleHack-Raleigh. See some features below:

  • Users sign up via Twitter
  • Use machine learning to find matches on JustGiving.com
  • Users start a physical challenge, invite friends, and make a pledge for a charity
  • Payments can be done via Paypal
  • Organizations engage with their donors via GotoWebinar, and they are able to learn about the work done by their favorite charities.
